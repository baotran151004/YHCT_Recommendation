# Traditional Vietnamese Medicine Dataset

This dataset supports the hybrid recommendation system proposed in our study.

## Tables

- symptoms.csv: list of clinical symptoms
- symptom_aliases.csv: mapping of synonym expressions
- syndrome_patterns.csv: syndrome definitions
- formulas.csv: herbal prescriptions
- formula_components.csv: composition of formulas
- pattern_principles.csv: diagnostic principles
- therapeutic_principles.csv: treatment strategies

## Description

The dataset models relationships between symptoms, patterns, and formulas to support clinical reasoning in Traditional Vietnamese Medicine.

## Usage

These CSV files can be imported into a PostgreSQL database or used directly for data processing and experimentation.
